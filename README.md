# Projeto Contador de Horas Extras - Base 027
Este repositório contém um projeto desenvolvido em linguagem C no contexto acadêmico da Universidade Vila Velha (UVV), em parceria com a empresa Base 027. Durante o desenvolvimento do projeto, foram utilizadas ferramentas de Inteligência Artificial, como ChatGPT e GitHub Copilot, como apoio ao processo de aprendizagem e à resolução de dúvidas técnicas.

O projeto consiste na implementação de um sistema de contagem e controle de horas extras, com o objetivo de aplicar, de forma prática, os fundamentos da programação e da lógica computacional estudados ao longo da disciplina, mantendo o protagonismo do desenvolvimento por parte do estudante.
